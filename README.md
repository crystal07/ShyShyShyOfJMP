Narsha Project Team (ShyShyShy Of JeongMin_Park)

# Project : Cafe & Bar
간단한 홈메이드 음료수들의 레시피를 알려줘 사용자들이 간단한 음료들을 직접 만들어 먹을 수 있게 해주는 서비스입니다.

# Project Function
 1. 베이스(소주/커피/그외)별로 칵테일 분류
 2. 음료의 이름(사진)을 보여줌
 3. 음료의 레시피를 보여줌
